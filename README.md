# Link Manager

A private, dark-themed dashboard for managing your important local and web links.

This release includes two versions of the application:

### 1. Windows Installer (`LinkManager_Setup.exe`)
*   **Best for:** Standard installation.
*   **How to use:** Download and run the executable. It will install Link Manager to your system, create a Start Menu shortcut, and an uninstaller.
*   **Data Location:** Data is stored in the application directory.

### 2. Portable Version (`LinkManager_Portable_Release.zip`)
*   **Best for:** USB drives or trying the app without installing.
*   **How to use:** 
    1. Download the ZIP file.
    2. Extract **ALL** contents to a folder.
    3. Run `LinkManager.exe` inside the folder.
*   *Note: Do not run directly from the zip without extracting.*

## ðŸ“– Detailed Usage Guide

### 1. Main Dashboard
The main view displays all your categorized links. Click any link card to open it.
*   **Filter:** Use the search bar at the top to filter links instantly.
*   **Manage Mode:** Click the **"Link Manager"** title in the header to enter the editing mode.

### 2. Managing Categories & Links
Once in Management Mode, you have full control over your data.
*   **Add Category:** Click the **"+ New Category"** button at the top right.
*   **Edit Category:** Click the pencil icon next to a category name to rename it.
*   **Add Link:** Click **"+ Add Link"** at the bottom of any category.
*   **Reorder:** Use the â–² and â–¼ arrows to move categories and links up or down.
*   **Delete:** Use the red trash bin icon to remove items.

<img width="1920" height="1032" alt="image" src="https://github.com/user-attachments/assets/cb4cd49b-d795-4575-91db-b3bb6c3510ea" />


### 3. Managing Tags
Customize the colored tags (e.g., "Social", "Dev", "Docs") to fit your workflow.
1.  Click the **"Manage Tags"** button in the header.
2.  **Add New Type:** Enter an ID (lowercase), a Label, and pick a color.
3.  **Delete Type:** Click the 'x' next to existing types to remove them.

<img width="466" height="592" alt="image" src="https://github.com/user-attachments/assets/7b09c8d0-50a2-4d92-b408-37f136cf8ed2" />


### 4. Backup & Restore
Your data is stored locally. To prevent data loss or move to another computer:
*   **Export:** Click the **"Export"** button to save your entire configuration as a `.json` file.
*   **Import:** Click **"Import"** to load a previously saved configuration. You can choose to **Merge** with existing data or **Replace** it entirely.
